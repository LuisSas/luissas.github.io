Comenzando con Coffeescript

Breve introducción

Hace poco tiempo estuve en el desarollo de un sistema informatico en el cual todo el frontend estaba hecho con coffeescript, por lo cual tuve la necesidad de aprender un poco acerca de este lenguaje.

Coffeescript es un transpilador de código, y esto que significa, que al momento de compilar su código este saca simplemente código de javascript, esto es bastante util y de gran utilidad ya que tendremos un mejor control de nuestros archivos.

El código que esta hecho con coffeescript es de muy facil escritura, al igual que el del lectura, este código se basa mucho en el interliniado, esto propicia a que tu código luzca limpio.

Instalación

La forma de instalar coffeescript en una computadora se hace de una forma muy sencilla, muy simple; Antes que nada tenemos que tener instalado node y el gestor de paquetes de este (npm)

npm install -g coffeescript

Con esto nosotros seremos capaces de usar coffeescript de una manera global en nuestra computadora, si queremos hacerlo a nivel de proyecto solo le quitamos la sentencia -g esto es lo que lo hace general.

Uso

Para poder hacer uso de este con teclear en la terminal coffee ingresamos a la shell de coffee con la que ṕodremos correr nuestros comandos o pequeños algoritmos hechos con coffee aparte de pequeñas ventajas como el autocompletado de código.

Compilación

Como se menciono anteriormente, lo que hace coffeescript es compilar a código de javascript, y para esto se hace con el comando de:

coffee -c file.coffee

Esto nos proporcionaria un archivo file.js en el cual veremos el archivo que nos compila coffee con una pequeña leyenda que nos describe la versión de coffeescript usada

coffeescript javascript
file.coffee file.js

Conclusión

Como se puede ver coffee es de mucha ayuda para nuestros proyectos, su uso es bastante sencillo y la forma de compilación es bastante util ya que aunque no se tenga el conocimiento de javascript de una inicio viendo los archivos compilados de coffee, se puede saber como hacer diferentes cosas de una manera mas sencilla, aunque no nos tenemos que olvidar de la parte mas importante “Coffeescript es solamente Javascript”
