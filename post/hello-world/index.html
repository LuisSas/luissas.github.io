Mi primer post… mi primera semana

Antes que nada en esta primera entrada les comentare de mi primera semana en MakinDevs, y como conoci (git) y (groovy).

El primer día de esta semana aprendí sobre Git, una herramienta muy util y agradable, al principio fue confuso ya que no tenia experiencia usando este tipo de herramientas \Un controlador de versiones o sistema manejador de archivos) pero conforme fue pasando el curso me pude dar cuenta de lo importante que era el ir usando este tipo de herramientas en proyectos colaborativos, o en proyectos personales.

En mi experiencia personal, al momento de hacer un proyecto en la escuela, nos dividiamos las partes que cada quien iba a hacer, y al momento de tenerlas recurriamos a usar una USB y todo lo juntabamos en las maquinas de los otros compañeros, Si hubieramos conocido GitHub desde un inicio nos pudimos haber organizado de una manera mejor y sobre todo hacer los proyectos mas rapido.

Curso de Git

Este fue mi primer acercamiento a Git, Git es un sistema de archivos el cual te permite el control de versiones de un proyecto o simplemente de un archivo, vimos los comandos bàsicos en la terminal desde como iniciar git en un lugar, hasta hacer nuestros primeros commits al igual que regresarnos en cambios que habíamos hecho.

Como dije anteriormente, me fue complicado en un inicio entender lo que era Git y como trabajaba, pero la forma en la que lo explico Juan fue bastante agradable y sencilla para que pudieramos entender bien de que trataba.

Mis primeras experiencias en Groovy

Esta semana tambien conoci Groovy, es un lenguaje muy potente y de facil manejo… nunca habia manejado un lenguaje con una sintaxis tan facil como este, por lo que entendi es un lenguaje derivado de java, en el cual puedes programar a la tipo java pero tambien con su propia sintaxis

Para terminar esta semana

Tuvimos el curso intermedio de Git, vimos comandos ya un poco mas avanzados, como “git merge” para poder juntar branches y algo sobre git rebase, estas partes fueron mas dificiles de entender, pero con la practica todo saldra bien.
